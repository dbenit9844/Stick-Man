Documentation:

Thank you for purchasing the 2D Stickman Action Combat Platformer Animation Pack (40+ Animations) by Monish Create.

This pack includes a collection of handcrafted 2D stickman animations designed for action, combat, and platformer games. All animations are provided as sprite sheets and Unity-ready animation clips.

Included in This Pack:
- 40+ handcrafted animations:
  * Idle variations
  * Walk, run, jump, fall variations
  * Dash, roll, slide
  * Player Attack combo
  * Telegraphed Enemy Attack combo
  * Hit reactions
  * Special poses & taunts
- Sprite sheets for each animation
- Unity animation clips (.anim)
- Demo scene
- Basic controller script for sequential preview

How to Use:
1. Import the .unitypackage into Unity.
2. Open the Demo Scene (optional) to preview animations.
3. Drag animation clips into your Animator Controller.
4. Adjust sprite and import settings based on your project.

IMPORTANT: Can change color of stickman simply by using the Unity SpriteRenderer Component
